About Blender

Blender is a free and open-source 3D computer graphics software toolset used for creating animated films, visual effects, art, 3D printed models, motion graphics, interactive 3D applications, and computer games.

Features

Blender includes numerous features which can be used for rendering, modelling, sculpting, 3D animation and rigging, VFX and even 2D animation.

Short Film

Before the release of the latest update, Blender 2.8, the company released a short film called "Spring" made entirely in the program.
